Patterns

Key CSS patterns used across the site

Section Container

/* Every section follows this wrapper pattern */ .section { width: 100%; padding-inline: var(--page-padding); /* 80px */ } .section .inner { max-width: var(--content-width); /* 1440px */ margin: 0 auto; padding-block: var(--section-padding); /* 120px */ }

Full-Bleed Right

/* Content overflows right edge, stays aligned left */ .full-bleed-right { display: grid; grid-template-columns: 1fr 1fr; gap: var(--space-40); } .full-bleed-right .media { margin-right: calc(-1 * var(--page-padding)); border-radius: 12px 0 0 12px; overflow: hidden; }

Hover Underline

/* Animated underline on hover via ::after */ .hover-underline { position: relative; } .hover-underline::after { content: ''; position: absolute; bottom: 0; left: 0; width: 0; height: 1px; background: currentColor; transition: width 0.3s ease; } .hover-underline:hover::after { width: 100%; }

Responsive Breakpoints

/* Tablet */ @media (max-width: 1024px) { .section { padding-inline: 40px; } } /* Mobile */ @media (max-width: 767px) { .section { padding-inline: 20px; } .section .inner { padding-block: 60px; } }
